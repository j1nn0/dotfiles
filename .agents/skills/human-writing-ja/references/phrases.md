# 注意フレーズ

## 多用に注意

### 「これにより」

多用すると機械的な印象になる。

| 例 | 簡潔な代替 |
|---|----------|
| 設定を変更しました。これにより、速度が向上します | 設定を変えたら速度が上がります |
| APIを導入しました。これにより、連携が可能になります | API導入で連携できるようになりました |

代替表現: 「すると」「その結果」「だから」、または接続詞を省いて直接つなげる

### 「～することができます」

短くできる場合は短くする。

| 例 | 簡潔な代替 |
|---|----------|
| 設定を変更することができます | 設定を変えられます |
| カスタマイズすることができます | カスタマイズできます |

### 「～において」「～における」

硬い印象になる。「〜で」で済む場合は置き換える。

| 例 | 簡潔な代替 |
|---|----------|
| ビジネスにおいて重要 | ビジネスで重要 |
| 日本における導入事例 | 日本での導入事例 |

## 冒頭の定型句

文章や段落の最初で避けるべき表現。

| 禁止 | 理由 | 代替 |
|-----|-----|-----|
| ～について説明します | 予告不要 | 削除して本題から |
| ～についてご紹介します | 同上 | 削除 |
| ～を深掘りしていきましょう | AI臭い | 削除 |
| 以下に示すように | 読めばわかる | 削除 |
| 上記の通り | 読めばわかる | 削除 |
| まず、次に、最後に | 機械的 | 省略か別の接続詞 |
| 結論から言うと | 言っておいて曖昧な結論が続く | 削除して直接結論を書く |
| 今日の～において | 時代背景から始める癖 | 削除 |
| 現代社会では | 同上 | 削除 |
| 急速に変化する現代において | 同上 | 削除 |

## 曖昧な量詞・修飾語

具体的な数字や例に置き換える。

| 禁止 | 問題点 | 代替例 |
|-----|-------|-------|
| さまざまな | 何がさまざまか不明 | 具体的に列挙 |
| 多くの | 何個か不明 | 「42社が」「8割が」 |
| あらゆる | 本当にすべてか | 限定して書く |
| 非常に | 程度が曖昧 | 具体的な数字 |
| とても | 同上 | 削除か具体化 |
| 大変 | 同上 | 削除か具体化 |
| かなり | 同上 | 削除か具体化 |
| 非常に重要 | 全部重要になる | 理由を書く |

## 大げさな形容詞

褒めすぎない。具体的な数字や事実で示す。

| 禁止 | 問題点 |
|-----|-------|
| 素晴らしい | 何が素晴らしいのか |
| 革新的な | 何が革新なのか |
| 画期的な | 同上 |
| 飛躍的な | 具体的な数字で示す |
| 圧倒的な | 同上 |
| 抜本的な | 同上 |
| 劇的な | 同上 |

## ヘッジ表現

自信がないように見える。事実なら断定する。

| 禁止 | 代替 |
|-----|-----|
| ～と言えるでしょう | ～です |
| ～かもしれません | 事実なら断定、不明なら書かない |
| ～と思われます | ～です / 主語を明確に |
| ～ではないでしょうか | 断定するか削除 |
| ～の可能性があります | 確率を示すか断定 |
| おそらく | 根拠を示すか削除 |
| 基本的に | 例外があるなら明示、なければ削除 |
| 一般的に | 出典を示すか削除 |

## 丁寧すぎる表現

多用すると冗長になる。必要な場面だけ使う。

| 注意 | 簡潔な代替 |
|-----|----------|
| ～させていただきます | ～します |
| ～していただければ幸いです | ～してください |
| ご確認いただけますと幸いです | 確認してください |
| ぜひ | 省略可 |
| ご安心ください | 省略可 |
| お気軽に | 省略可 |
| よろしければ | 明確に提案する |

## 強調パターン

本当に重要なら読めばわかる。

| 禁止 | 代替 |
|-----|-----|
| 重要なのは～ | 削除してそのまま書く |
| 注目すべきは～ | 同上 |
| 特筆すべきは～ | 同上 |
| ここで重要なのが～ | 同上 |
| ポイントは～ | 同上 |
| 重要なポイントとして | 同上 |
| ～が鍵となります | 理由を書く |

## 接続詞の機械的使用

同じ接続詞を連続で使わない。

| 問題パターン | 対策 |
|------------|-----|
| しかし、しかし... | 2回目以降は「でも」「だが」に変えるか削除 |
| また、また... | 「さらに」「加えて」に変えるか削除 |
| そして、そして... | 削除 |
| 一方で... 他方で... | 片方削除 |
| つまり（多用） | 削除して直接説明 |
| したがって（多用） | 「だから」「なので」 |

## 指示代名詞の過剰使用

英語のit/thatの直訳。日本語では省略するか具体的な名詞に置き換える。

| 問題 | 改善 |
|-----|-----|
| それは重要です | 削除して前文と直接つなげる |
| その結果 | 「すると」か削除 |
| このことから | 削除 |
| これらの | 具体的に何かを書く |
| そのため | 「だから」か削除 |

## ビジネス用語・カタカナ語の乱用

日本語で言えることはそちらで。

| 禁止 | 代替 |
|-----|-----|
| ソリューション | 解決策、やり方 |
| アプローチ | 取り組み、方法 |
| インサイト | 気づき、発見 |
| レバレッジ | 活用（そもそも避ける） |
| シナジー | 相乗効果（そもそも避ける） |
| エコシステム | 仕組み、環境 |
| 最適化 | 改善、調整 |
| ベストプラクティス | 定石、お手本 |
| フレームワーク | 枠組み |
| コンセプト | 考え方 |
| ポテンシャル | 可能性 |
| バリュー | 価値 |

## 文末の定型パターン

同じ文末が続くと単調になる。文体を変える必要はなく、同じ文体の中で語尾を変えればよい。

### ですます調の場合

| 問題 | 対策 |
|-----|-----|
| ～です。～です。～です。 | 「～でした」「～になります」「～でしょう」を混ぜる |
| ～ます。～ます。～ます。 | 「～ました」「～ています」「～ません」を混ぜる |
| ～しましょう。～しましょう。 | 「～してください」「～できます」に変える |

### だ/である調の場合

| 問題 | 対策 |
|-----|-----|
| ～だ。～だ。～だ。 | 「～である」「～になる」「体言止め」を混ぜる |
| ～である。～である。～である。 | 「～だ」「～となる」「体言止め」を混ぜる |

### 共通

| 問題 | 対策 |
|-----|-----|
| ～が重要です。～が大切です。～が必要です。 | 同義反復。1つに絞り、理由を添える |
